    • CV INTERACTIF

    J’ai réalisé ce projet durant ma première année de DUT MMI. Le CV numérique est un projet étudiant dans le cadre de la composante “intégration web”. Ce projet consiste à réaliser un site web interactif présentant notre CV classique pdf. Le but est de rendre son CV plus original et plus ludique pour le recruteur. Ce que j’ai le plus aimé faire dans ce projet est de réfléchir à une navigation originale. Ce projet m’a apporté de l’assurance et des compétences techniques supplémentaires en HTML, CSS, JavaScript et PHP.

    QUORIDOR

    J’ai réalisé ce projet durant ma deuxième année de DUT MMI. Le Quoridor est un projet étudiant dans le cadre de la composante programmation orientée objet. Le but de ce projet, composé de deux étudiants, consiste à réaliser une page html permettant de jouer au jeu de plateau Quoridor à deux joueurs. Le développement de ce jeu se fait principalement en JavaScript et jQuery. Dans mon groupe, je m’occupe de toute la partie code tandis que mon duo s’occupe de la partie design du jeu.

    BOOSKA-BÊTA

    Booska-Bêta est mon projet tutoré de deuxième année de DUT MMI. Je fais partie d’une équipe de 5 étudiants, sous le nom de Booska-Bêta, qui s'inscrit dans le cadre d'un projet client visant à réaliser la refonte du site actuel du média Booska-P. Je travaille en tant que développeur full-stack dans une équipe de 5 étudiants et je collabore avec les acteurs et fondateurs du média. Le but de ce projet est d’avoir un projet concret avec la récompense de voir son travail aboutir et aider un des médias les plus influents sur Instagram en ce moment. Mon rôle au sein de l’équipe est de coder principalement la partie back de ce nouveau site web. Cliquez ici pour accéder au site.

    PORTRAIT CHINOIS

    J’ai réalisé le projet de portrait chinois durant ma première année de DUT. Le but du portrait chinois est d’utiliser des analogies du type "si j'étais..., je serais..." afin de se décrire personnellement à travers nos goûts, nos passions, ce que notre imagination nous donne, notre caractère. Ce projet consiste donc à faire un site web répertoriant 7 analogies différentes dans le cadre de la composante intégration web.

    BLOG

    J’ai réalisé ce projet durant ma deuxième année de DUT MMI. Ce blog est un projet étudiant dans le cadre de la composante développement web et php. Il consiste à réaliser un blog avec des sessions, une base de données, et un modèle CRUD, via les outils Doctrine et Composer principalement. Le développement se fait aussi en HTML, CSS, JavaScript et PHP. C’était un projet très enrichissant, à concevoir seul.

    COMBAT DE PERSONNAGES
